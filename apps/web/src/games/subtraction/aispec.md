# RestGame (Subtraction)

## What
Game for practicing subtraction by solving problems like "42 - 15 = _"

## Boundaries
- Uses global game engine for score/streak/level tracking
- Answer is always the subtraction result
- Difficulty increases by allowing larger numbers (level multiplied by 10)

## Success
- Player answers correctly within timeout
- Score increases, streak continues
- Incorrect answers reset streak and deduct points
