# DivisionGame

## What
Game for practicing division by solving problems like "49 ÷ 7 = _"

## Boundaries
- Uses global game engine for score/streak/level tracking
- Only generates problems with exact integer results (no remainders)
- Answer is always the quotient (whole number result)
- Difficulty capped at 12÷12 to keep it reasonable for kids

## Success
- Player answers correctly within timeout
- Score increases, streak continues
- Incorrect answers reset streak and deduct points
