# Musical Notation Learning Game

## Overview
An interactive game to learn musical notation through various mini-games. Users develop their ability to read, understand, and create musical notation.

## Current Mini-Games

### Echo
- **Description**: Listen to a rhythm pattern and repeat it using left (F) and right (J) hand keys
- **Mechanics**: 
  - Machine plays a 4/4 bar pattern
  - User must echo the pattern with proper timing and duration
  - Scoring based on timing accuracy (±150ms tolerance) and duration accuracy (±20% tolerance)
  
- **Levels**:
  1. **Level 1**: Single hand patterns with various note durations (whole, half, quarter, eighth, sixteenth notes)
  2. **Level 2**: Both hands playing identical patterns
  3. **Level 3**: Different patterns per hand simultaneously

- **Scoring**: 
  - 5 points for correct note timing
  - 5 points for correct note duration
  - Per hand in Level 1, user gets 10 free points for unused hand
  - 70% threshold needed to pass
